## SQL Analysis Summary

The analysis explores restaurant sales performance using SQL, focusing on:

- Menu structure and pricing analysis
- Category-level dish distribution and average pricing
- Order volume trends over time
- High-value customer orders and revenue concentration
- Most and least frequently ordered items
- Top revenue-generating menu items and orders
- Item and category breakdown for highest-spending orders

These insights help understand customer preferences, pricing strategy, and revenue drivers.

All analysis queries are available in the SQL analysis file.
